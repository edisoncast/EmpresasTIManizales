# Revisión de presencia web de empresas — 2026

## Alcance y método

Esta bitácora acompaña la revisión de las empresas del directorio que no tienen
sitio web ni perfil corporativo de LinkedIn. La fuente de Cámara de Comercio
confirma la existencia legal, pero no equivale a una URL pública de la empresa.

Cada registro se revisa mediante búsquedas por razón social, nombre comercial y
ubicación. Solo se publica una URL cuando se puede atribuir de forma clara a la
empresa. No se usan como enlaces corporativos los directorios de terceros,
perfiles personales ni dominios con homónimos no resueltos.

Estados:

- `sitio oficial confirmado`: se añadió al registro de datos.
- `sin URL confirmada`: la búsqueda no produjo un sitio o LinkedIn corporativo
  atribuible con seguridad; queda pendiente de una fuente aportada por la
  empresa o comunidad.
- `resultado no aplicable`: la coincidencia encontrada pertenece a otra entidad
  o no confirma un canal oficial.

## Próximo criterio de priorización

1. Empresas activas con tamaño pequeño, mediano o grande.
2. Microempresas con nombres comerciales distintivos.
3. Establecimientos de comercio y personas naturales, para los que se exigirá
   una coincidencia especialmente clara antes de publicar un enlace.
